Flusso temporale

t=0ms  Utente clicca lab → POST /api/lab/start parte in background
t=0ms  LabBootSplash appare, le righe si stampano una per una (~150ms gap)
t=~1s  Tutte le righe visibili + "press any key to enter"
t=?  Docker risponde con containerName (nel frattempo)
keypress  Se Docker è pronto → apre LabTerminal. Se non ancora → spinner breve
